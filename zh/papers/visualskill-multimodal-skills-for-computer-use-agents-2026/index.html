深度分析

创新点

  • 分层多模态技能（VISUALSKILL），在文本旁保留视觉图形，组织为按主题文件的中央索引，并通过load_topic MCP工具访问。
  • 两阶段技能构建流水线，结合编写文档与实时应用程序UI探索。
  • 证明多模态技能工件相比纯文本技能有显著提升，证据表明图形有助于识别UI元素和验证工作流状态。

方法

VISUALSKILL是一种针对每个应用定制的分层多模态技能，通过两阶段流水线（编写文档+实时UI探索）构建。代理使用load_topic MCP工具按需获取相关主题的文本和图形。在CUA-World和OSExpert-Eval基准上使用Claude Code CLI（Claude Opus 4.6）进行评估，比较无技能、纯文本技能和VISUALSKILL。

关键结果

VISUALSKILL平均得分为0.456，比无技能基线（0.303）绝对提升15.3个百分点，比匹配的纯文本技能（0.373）绝对提升8.3个百分点，表明保留视觉图形改善了UI元素识别和工作流状态验证。
