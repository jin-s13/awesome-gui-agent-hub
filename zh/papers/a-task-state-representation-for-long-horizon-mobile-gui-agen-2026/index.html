深度分析

创新点

  • 任务状态表示（TSR）框架，明确将持久任务状态与瞬时屏幕观察解耦
  • 三个结构化组件：全局指令摘要、子目标的动态进度跟踪器和过渡感知动作验证器
  • 无需训练的轻量级外部包装器，无需架构修改即可引导代理推理
  • 通过动作前后的视觉比较进行连续状态更新

方法

TSR是一种无需训练的外部包装器，维护全局指令摘要、动态进度跟踪器和过渡感知动作验证器，通过比较动作前后的截图进行更新，以引导代理的推理循环。它在四个移动GUI基准上进行了评估，无需改变模型架构。

关键结果

TSR在四个移动GUI基准上的复杂跨应用和内存密集型任务中，成功率提升了高达12个绝对百分点。
