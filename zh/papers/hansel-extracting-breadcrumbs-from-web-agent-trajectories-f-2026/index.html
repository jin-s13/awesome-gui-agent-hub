深度分析

创新点

  • 从Web智能体轨迹中提取交互式、可验证的证据页面和片段，作为可导航的面包屑
  • 在交互式视图中保留相关页面状态（应用的筛选条件、搜索查询和滚动位置）以提供上下文
  • 当智能体的答案无法追溯到任何访问过的页面时，明确标记这一缺口

方法

HANSEL是一个系统，它处理Web智能体轨迹以提取证据页面和片段，并将其呈现为具有保留状态的交互式视图。该系统在AssistantBench和Online-Mind2Web的45个任务上进行了技术评估，使用了精确率、召回率和轨迹体积缩减指标，并在一个包含14名参与者的受控用户研究中，将HANSEL与标准智能体界面在任务完成时间、感知努力、可用性、验证便捷性和错误识别方面进行了比较。

关键结果

技术评估显示精确率为83.7%，召回率为88.8%，轨迹体积缩减61.6%；用户研究表明任务完成时间和感知努力显著减少，在可用性、验证便捷性和错误识别方面的评分显著更高。
