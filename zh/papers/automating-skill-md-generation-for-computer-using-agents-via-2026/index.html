深度分析

创新点

  • 一种从GUI交互轨迹自动生成SKILL.md文件的三阶段流水线：分割、聚类和技能感知策略训练。
  • 一项诊断性研究，揭示挖掘出的技能聚类具有高度可读性（相对于真实标签的高纯度），但并不能可靠地改进下游策略性能，凸显了可检查性与可迁移性之间的差距。

方法

本文提出一个流水线，对GUI交互轨迹进行分割，将片段聚类为候选技能，并使用聚类标注训练技能感知策略。在InteraSkill Workflows和BrowseComp+基准上评估，使用GRPO微调策略，测量技能步骤准确率和源域指标。

关键结果

八个挖掘出的聚类中有五个相对于InteraSkill Workflows标签的纯度至少达到0.95，但GRPO仅将技能步骤准确率从18.5%提升至20.5%，对BrowseComp+无影响，并且在关键源域指标上不如简单的频率先验。

局限性

  • 当前的边界检测器不足以实现可靠的跨域策略改进。
  • 无序片段表示限制了可迁移性。
  • 离线奖励模型不足以实现策略改进。
  • 挖掘出的技能的可读性并不能保证改进下游策略性能。
