深度分析

创新点

  • 以真实设备为主的混合基础设施，将物理设备作为主要执行环境，沙箱作为辅助支持，使数据收集、训练、部署和评估的分布与真实部署一致。
  • 多源训练数据，涵盖高频头部任务、面向长尾意图的高泛化数据以及用于反思和记忆的能力增强数据。
  • 错误驱动的数据飞轮，将失败轨迹转化为纠正动作、反思解释和恢复演示。
  • 渐进式三阶段训练流程：监督微调、步骤级强化学习和智能体强化学习。

方法

Xiaomi-GUI-0是一个原生多模态GUI智能体，在真实设备闭环中进行训练和评估。它采用以真实设备为主的混合基础设施，物理设备为主要环境，沙箱为辅助支持。训练数据来自多个来源，并通过错误驱动的数据飞轮进行增强，模型通过监督微调、步骤级强化学习和智能体强化学习的渐进式流程进行训练。

关键结果

Xiaomi-GUI-0在内部RealMobile基准上达到72.0%的成功率，在AndroidWorld上达到78.9%，并在真实任务中显著提升了执行稳定性和异常状态识别能力。
