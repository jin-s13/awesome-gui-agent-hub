Reinforcement Learning for Computer-Use Agents with Autonomous Evaluation

arXiv 2026 65.8 method

TLDR

RL fine-tuning for computer-use agents using autonomous vision-language evaluation as a noisy reward signal, with noise correction improving success rates across three benchmarks.

Reasoning

The paper presents a novel RL framework that leverages VLM-based autonomous evaluation for reward, addressing a key challenge in GUI agent training. Strengths include explicit noise modeling and empirical validation on multiple real-world desktop benchmarks. Weaknesses: limited to desktop environments, no discussion of web/mobile agents, and reliance on VLM evaluator quality.

Deep Analysis

Innovations

  • Using autonomous vision-language evaluation as a scalable reward signal for RL fine-tuning of GUI agents, eliminating the need for handcrafted rewards or manual labels.
  • Modeling the evaluator's feedback as a noisy binary reward channel and deriving a noise-corrected reward estimator for Proximal Policy Optimization.
  • Demonstrating that noise-corrected evaluator rewards significantly outperform both zero-shot baselines and raw evaluator fine-tuning across multiple GUI benchmarks.

Methodology

The framework fine-tunes a GUI agent policy with RL using a Vision-Language Model (VLM) to judge task completion from a final screenshot and instruction, providing terminal binary feedback. Because the VLM evaluator is imperfect, its feedback is modeled as a noisy binary channel, and a noise-corrected reward estimator is derived for Proximal Policy Optimization (PPO). Experiments are conducted on macOSWorld, Windows Agent Arena, and OSWorld.

Key Results

Noise-corrected evaluator rewards improve success rates by an average of 12.6 percentage points over zero-shot baselines and 5.1 points over raw evaluator fine-tuning across the three benchmarks.

Limitations

  • Autonomous evaluators are imperfect, introducing noise into the reward signal that must be explicitly modeled and corrected.
