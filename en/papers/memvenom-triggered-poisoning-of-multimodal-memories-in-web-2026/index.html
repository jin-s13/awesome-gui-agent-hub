Deep Analysis

Innovations

  • Identification and systematic study of multimodal memory poisoning in web agents
  • MemVenom: a unified black-box attack framework poisoning graph-structured external memory with coordinated text-image evidence
  • Two-stage design: trigger-conditioned retrieval attack and post-retrieval attack induction via adversarial perturbations and stealthy OCR injection
  • Persistent, reusable, goal-agnostic attacks without model parameter modification or task re-optimization

Methodology

MemVenom is a black-box attack that poisons graph-structured external memory in web agents. It uses a two-stage process: a trigger-conditioned retrieval attack to ensure high-probability recall of malicious memory, and a post-retrieval attack induction that applies adversarial perturbations and stealthy OCR injection to override the original user objective. The method is evaluated across multiple web-agent frameworks and vision-language models.

Key Results

MemVenom achieves up to 99.15% attack success rate on GPT-5-family web agents with minimal impact on benign performance, and transfers effectively across architectures and model scales.
