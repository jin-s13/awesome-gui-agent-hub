Deep Analysis

Innovations

  • PageMem: a structured page representation deterministically constructed from the DOM, exposing each page as a hierarchy of semantic sections with short summaries.
  • Divide-and-conquer observation pipeline that skims section summaries and extracts details only from task-relevant regions.
  • Lightweight exploration and memory system that traverses each website once to build a reusable map of pages and element behaviors.
  • Compound action workflows that collapse common multi-step interactions into single agent actions, handling partial state changes automatically.

Methodology

WebChallenger is a web agent framework built on PageMem, a DOM-derived hierarchical representation of pages with semantic sections and summaries. It incorporates three mechanisms: a divide-and-conquer observation pipeline for selective attention, a one-time exploration and memory system for persistent website structure, and compound action workflows for procedural fluency. The system uses off-the-shelf open-weight models without fine-tuning and is evaluated on four web navigation benchmarks.

Key Results

WebChallenger achieves 56.3% on WebArena, 48.7% on VisualWebArena, 51.0% on Online-Mind2Web, and 70.9% on WorkArena, approaching the performance of frontier proprietary systems while being more cost-efficient.
