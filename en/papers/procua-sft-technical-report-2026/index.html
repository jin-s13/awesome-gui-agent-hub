Deep Analysis

Innovations

  • ProCUA-SFT dataset: 3.1M step-level SFT samples distilled from 93K synthetic trajectories across 2,484 application combinations, addressing negative transfer from human-collected data.
  • Fully automated pipeline that synthesizes grounded tasks on live desktops using real-world content (spreadsheets, presentations, OSWorld configs) and verifies task feasibility via binary precondition checking.
  • Single VLM (Kimi-K2.5) serving as goal generator, precondition judge, and trajectory executor, eliminating planner-actor capability gaps.
  • Step-prefix sample expansion that exactly reproduces the context layout seen at inference time.
  • Demonstrated large improvement: fine-tuning UI-TARS 7B on ProCUA-SFT yields 45.0% on OSWorld, an 18.7 point gain over base model and over 35% above AgentNet-trained counterparts.

Methodology

The authors create ProCUA-SFT, a dataset of 3.1M step-level SFT samples from 93K synthetic trajectories generated by an automated pipeline. The pipeline synthesizes tasks on live desktops seeded with real-world content (spreadsheets, presentations, OSWorld configs), checks task feasibility via binary precondition verification, and uses a single VLM (Kimi-K2.5) for goal generation, precondition judging, and trajectory execution. Each trajectory is expanded into step-prefix samples that match inference-time context layout, and the dataset is used to fine-tune UI-TARS 7B.

Key Results

Fine-tuning UI-TARS 7B on ProCUA-SFT for one epoch achieves 45.0% success rate on OSWorld, an 18.7 percentage-point improvement over the base model (26.3%) and over 35% above models fine-tuned on AgentNet (8-10%). A subset of ProCUA contributed to the computer-use capabilities of Nemotron 3 Nano Omni.
