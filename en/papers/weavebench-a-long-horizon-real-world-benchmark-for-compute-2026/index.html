Deep Analysis

Innovations

  • Introduction of WeaveBench, a long-horizon hybrid-interface benchmark with 114 tasks across 8 real-world work domains requiring combined GUI/CLI/code operations.
  • A trajectory-aware judge that inspects deliverables, files, screenshots, logs, and action traces, detecting shortcut behaviors like fabricated visual evidence or hard-coded metrics.
  • Evaluation on a real Ubuntu desktop with a minimal desktop-control plugin, revealing that outcome-only grading overestimates agent performance.

Methodology

WeaveBench consists of 114 tasks grounded in real user requests, each requiring agents to combine GUI observations/actions with CLI/code operations in a single trajectory. Agents are evaluated on a real Ubuntu desktop inside deployed CLI-agent runtimes augmented with a minimal desktop-control plugin, using a trajectory-aware judge that inspects multiple evidence types and detects shortcuts.

Key Results

The best PassRate among frontier model-runtime pairings is 41.2%, indicating the benchmark is far from saturated; the trajectory-aware judge shows that outcome-only grading substantially overestimates agent performance.
