Deep Analysis

Innovations

  • Proposes the 'Know Deeply, Act Perfectly' paradigm that unifies cognitive comprehension and operational execution by leveraging accumulated user interaction and task-running experience to directly improve accuracy and efficiency.
  • Introduces KnowAct-GUIClaw, a Know-Route-Act-Reflect framework that addresses OpenClaw's cross-platform GUI interaction deficits and lack of recursive self-improvement.
  • Features a pluggable GUI subagent with an experience-attributable memory system (Know) and a self-evolving skill library (Act) for seamless cross-platform migration and fast-path integration.
  • Continuously stores user profiles and feedback to enhance the accuracy of task decomposition and tool calls over time.
  • Demonstrates that the framework's memory and execution skills are transferable across diverse base models, improving Kimi-2.6 performance by 8.5%.

Methodology

The KnowAct-GUIClaw framework consists of a host agent that uses accumulated experience for long-horizon task decomposition and a pluggable GUI subagent equipped with an experience-attributable memory and self-evolving skill library, continuously integrating user profiles and feedback. It is evaluated across Android, iOS, HarmonyOS, and Windows on the long-horizon MobileWorld benchmark, comparing against other agentical frameworks and closed-source models like Seed-2.0-Pro and GPT-5.5.

Key Results

KnowAct-GUIClaw with open-source Kimi-2.6 achieves 64.1% on MobileWorld, outperforming all agentical frameworks and closed-source models such as Seed-2.0-Pro and GPT-5.5, and its memory/skills transfer yields an 8.5% improvement for Kimi-2.6.
