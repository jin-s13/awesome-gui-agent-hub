Deep Analysis

Innovations

  • Skill boundary that leverages multi-source supervision signals to identify safe skills and construct self-improving safety constraints
  • Selective skill reuse with context-guided task decomposition and selective activation of skill subsets
  • Modeling skill learning and utilization as a safety-constrained interaction process

Methodology

SkillHarness models skill learning and utilization as a safety-constrained interaction process. It introduces a skill boundary using multi-source supervision signals to identify safe skills from trajectories and build self-improving safety constraints. It also employs selective skill reuse, where tasks are decomposed by context and completed via selective activation of skill subsets.

Key Results

SkillHarness reduces the unsafe rate of learned skills by 57.1% and consistently improves execution stability under dynamic environmental changes, outperforming existing baselines.
