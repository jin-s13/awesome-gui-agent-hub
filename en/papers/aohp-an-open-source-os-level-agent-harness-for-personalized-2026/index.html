TLDR

AOHP is an open-source OS-level agent harness on Android that treats agents as first-class OS actors, improving task completion, cost, and security.

Reasoning

The paper presents a novel system design for agent-native operating systems with clear empirical improvements in task completion rate, token cost, and security compliance. However, it is limited to Android and preliminary experiments may not generalize to all OS agent scenarios.

Deep Analysis

Innovations

  • Open-source OS-level agent harness (AOHP) built on AOSP, providing a testbed for agent-native OS research
  • Treating agents as first-class OS actors with adaptive user interfaces and agent-friendly runtime environments
  • Personalized service composition mechanism
  • Efficient agent interfaces mechanism
  • Secure information flow mechanism

Methodology

AOHP is implemented on the Android Open Source Project, introducing three agent-oriented system mechanisms: personalized service composition, efficient agent interfaces, and secure information flow. Preliminary experiments were conducted on challenging tasks covering key OS agent capabilities, comparing task completion rate, execution token cost, and security-policy compliance against a baseline.

Key Results

AOHP achieved a 21.12% higher task completion rate, 51.55% lower execution token cost, and improved security-policy compliance compared to the baseline.

Limitations

  • Preliminary experimental evaluation only
  • Limited to the Android ecosystem (AOSP-based)
