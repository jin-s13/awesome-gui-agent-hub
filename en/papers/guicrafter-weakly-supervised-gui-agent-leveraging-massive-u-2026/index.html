GUICrafter: Weakly-Supervised GUI Agent Leveraging Massive Unannotated Screenshots

arXiv 2026 71.6 method

TLDR

GUICrafter uses weakly-supervised learning from unannotated screenshots to train GUI agents, achieving strong performance with minimal annotated data.

Reasoning

The paper introduces a novel curriculum learning approach that leverages massive unannotated screenshots for visual grounding, then fine-tunes with small annotated data via reinforcement learning. Strengths include data efficiency and competitive results; weaknesses include lack of explicit real-world benchmark details in the abstract and potential reliance on specific webpage data.

Deep Analysis

Innovations

  • Weakly-supervised GUI agent leveraging massive unannotated screenshots to reduce reliance on expensive human annotations
  • Curriculum learning framework with two stages: visual grounding from unannotated data, then calibration with small high-quality data via reinforcement learning
  • Achieves competitive performance to UI-TARS with only 0.1% of its data

Methodology

GUICrafter uses a two-stage curriculum: first, the model learns visual grounding from large-scale unannotated screenshots and webpages using contextual signals without human annotations; second, it is calibrated with a small amount of high-quality annotated data via reinforcement learning.

Key Results

GUICrafter achieves competitive or superior performance to UI-TARS while using only 0.1% of its data, and surpasses previous methods like GUI-R1 under the same amount of annotated data.
