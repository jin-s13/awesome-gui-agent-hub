Deep Analysis

Innovations

  • Taxonomy of four GUI agent failures (cognitive failure, visual state misunderstanding, hidden operation blindness, grounding error) mapped to perception-reasoning-action pipeline
  • Action-Grounded Visual Memory (AGMem) framework that stores image crops of local GUI regions related to successful actions or recoveries instead of full screenshots

Methodology

The paper introduces a taxonomy of GUI agent failures and analyzes the effect of full-image visual memory on failure distribution. It then proposes AGMem, which stores action-grounded image crops, and evaluates on the OSWorld benchmark.

Key Results

Full-image memory reduces state-level failures but worsens action-level ones, increasing hidden operation blindness and grounding error. AGMem improves task success rates by 33.3% over full-image memory.
