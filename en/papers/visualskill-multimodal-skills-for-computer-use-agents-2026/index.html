Deep Analysis

Innovations

  • Hierarchical multimodal skill (VISUALSKILL) that retains visual figures alongside text, organized as a central index over per-topic files and accessed via a load_topic MCP tool.
  • Two-stage skill construction pipeline combining authored documentation with live-application UI exploration.
  • Demonstration that multimodal skill artifacts yield significant gains over text-only skills, with evidence that figures help identify UI elements and verify workflow state.

Methodology

VISUALSKILL is a hierarchical multimodal skill tailored per application, built by a two-stage pipeline (authored documentation + live UI exploration). The agent uses a load_topic MCP tool to fetch relevant topic's text and figures on demand. Evaluation on CUA-World and OSExpert-Eval benchmarks with Claude Code CLI (Claude Opus 4.6) comparing no-skill, text-only skill, and VISUALSKILL.

Key Results

VISUALSKILL achieves an average score of 0.456, a +15.3 point absolute lift over the no-skill baseline (0.303) and +8.3 points over a matched text-only skill (0.373), showing that retaining visual figures improves UI element identification and workflow state verification.
