Speculative Rollback Correction for Quality-Diverse Web Agent Imitation

arXiv 2026 61.5 method

TLDR

Proposes Speculative Rollback Correction, a branch-level imitation framework for web agents that uses fixed-horizon review and quality-diversity archive to improve error recovery and trajectory diversity.

Reasoning

The paper introduces a novel training framework (SRC) that addresses the intervention timing problem in imitation learning for web agents, with clear methodology and results on WebArena-Infinity. Strengths include the quality-diversity archive and fixed-horizon review; weaknesses include reliance on resettable environments and evaluation on a single benchmark.

Deep Analysis

Innovations

  • Speculative Rollback Correction (SRC) framework with fixed-horizon branch review instead of step-level or trajectory-level correction
  • Teacher localizes the first harmful deviation only when local progress breaks, enabling rollback to preserve useful prefixes
  • Quality-diversity archive that retains verifier-passing trajectories from successful speculative rollouts
  • Next-action supervised fine-tuning on both localized corrections and verifier-passing trajectories

Methodology

SRC is a branch-level imitation framework for resettable environments where the student executes a short speculative segment, the teacher reviews and identifies the first harmful deviation when progress breaks, and rollback preserves correct prefixes. Successful rollouts are filtered by a hard verifier and stored in a quality-diversity archive, and the collected data is used for next-action supervised fine-tuning.

Key Results

On WebArena-Infinity, SRC collected 977 verifier-passing trajectories and 9,183 next-action examples, and fixed-horizon review improved the recovery-versus-query tradeoff over step-level review while retaining solution variants.
