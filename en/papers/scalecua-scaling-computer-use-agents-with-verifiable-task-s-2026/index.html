Deep Analysis

Innovations

  • VeriGen: an end-to-end framework for generating verifiable RL tasks via iterative docker interactions and a multi-agent feedback loop
  • Frontier Sampling: a method that tracks per-task capability and allocates rollouts to the current learning frontier to maximize sample efficiency
  • Visual Context Segmentation: a sliding window over recent visual context that balances rollout and training-engine pressure, yielding a 2.83x training speedup

Methodology

ScaleCUA integrates VeriGen for verifiable task synthesis using 100+ concurrent agents and a multi-agent feedback loop, producing 24K+ tasks. It then applies online RL with Frontier Sampling for efficient rollout allocation and Visual Context Segmentation for faster training, benchmarking on OSWorld and ScienceBoard.

Key Results

ScaleCUA achieves 68.7% on OSWorld and 54.0% on ScienceBoard, setting new state-of-the-art among open-source computer use agents, with a 2.83x training speedup.
