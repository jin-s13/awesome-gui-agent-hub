Deep Analysis

Innovations

  • FaraGen1.5: a scalable data pipeline with modular components (environments, solvers, verifiers) for computer use agent training
  • Synthetic environments that simulate domains gated by authentication or requiring irreversible actions, complementing live websites
  • Solver harness supporting multiple models (including GPT-5.4) and a user simulator for multi-turn rollouts
  • Three complementary verifiers scoring trajectories on task correctness, efficiency, and critical-point adherence
  • Training recipe balancing pipeline data for broad coverage, high-value tasks, and iterative correction of model deficiencies
  • Fara1.5 family of native computer use agents (4B, 9B, 27B) setting new state-of-the-art for their size class

Methodology

FaraGen1.5 generates training data using live and synthetic environments, a solver harness with multiple models and a user simulator, and three verifiers. The resulting data is used to supervised-finetune Fara1.5 agents (built on Qwen3.5 at 4B, 9B, 27B) with a balanced recipe covering broad coverage, high-value tasks, and iterative deficiency targeting. Agents are evaluated on browser-use benchmarks Online-Mind2Web and WebVoyager.

Key Results

Fara1.5-9B achieves 63.4% on Online-Mind2Web and 86.6% on WebVoyager; Fara1.5-27B reaches 72.3% on Online-Mind2Web, competitive with much larger proprietary systems, and each model sets a new state of the art for its size class.
