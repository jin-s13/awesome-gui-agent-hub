Deep Analysis

Innovations

  • Visual process reward framework comparing pre-action and post-action screenshots in visual feature space
  • Siamese vision transformer for change-aware representations
  • Action-Aware Critic Head jointly evaluating action success, task progress, and error type
  • Weakly supervised critic-training data construction pipeline from existing trajectories without additional human labels

Methodology

VisCritic uses a Siamese vision transformer to extract change-aware representations from pre- and post-action screenshots, and an Action-Aware Critic Head to predict action success, task progress, and error type. Training data is generated via a weakly supervised pipeline from existing agent trajectories without extra human labels, and the model serves as a plug-and-play process reward for GUI agents.

Key Results

Across five benchmarks, VisCritic generally improves benchmark metrics for diverse GUI agents and provides visual diagnostic cues.
