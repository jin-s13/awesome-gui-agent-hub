A Learning-Rate-Gated Failure of GRPO in a Small Language and Vision-Language Model Web Agent: A Controlled Null and Its Mechanism

arXiv 2026 61.4 method, theory

TLDR

GRPO fails to improve a small web agent on mastered tasks; only helps when there is headroom, with learning rate gating failure.

Reasoning

The paper's strength is its rigorous controlled experiments and null hypothesis testing across multiple conditions, clearly showing GRPO's failure to improve a strong baseline. A weakness is its limited scope to small models (4B-8B) and tasks already mastered, leaving open questions about more challenging settings.

Deep Analysis

Innovations

  • Controlled grid of 18 GRPO runs showing no credible improvement over a strong supervised web agent baseline at 4B–8B scale
  • Demonstration that GRPO only helps when sampled policy already outperforms the greedy one, i.e., when there is headroom to climb
  • Double-dissociation mechanism: moderate learning rates degrade the agent via attention/MLP blocks, high learning rates cause collapse not traceable to any single group, and embedding changes are causally inert
  • Scale-dependent coupling between effective rank in late layers and capability, present at 4B but not at 8B

Methodology

The study runs a controlled grid of 18 GRPO experiments varying learning rate, KL weight, seed, initialization, and clipping on a 4B–8B language and vision-language web agent, using both text and Set-of-Marks screenshot observations. It employs paired testing with 25 evaluation seeds and 6 training seeds, and validates the pipeline with a control task where reward is reachable by sampling, then uses grafting and effective rank analysis to dissect the mechanisms of failure.

Key Results

No GRPO configuration credibly improves the supervised baseline; moderate to high learning rates credibly degrade the text track, while the null holds under Set-of-Marks and scaling to 8B. GRPO succeeds only when the sampled policy already outperforms the greedy one, and the failure exhibits a double dissociation: degradation localizes to attention/MLP blocks, collapse is not group-specific, and embedding changes are causally inert; effective rank tracks capability at 4B but diverges at 8B.

Limitations

  • Results are specific to the 4B–8B scale and may not generalize to larger models
  • Findings are limited to the web agent domain and a strong supervised baseline
  • Credible harm is a text-track finding and only nominal under Set-of-Marks observations
  • The scale-dependent effective rank coupling is reported for the smaller model only
