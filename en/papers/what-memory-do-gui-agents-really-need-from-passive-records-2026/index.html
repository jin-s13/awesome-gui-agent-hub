Deep Analysis

Innovations

  • Active Task Driving Memory (ATMem): shifts GUI-agent memory from passive storage to an actively maintained execution state that links each value to its role and current status.
  • STR-GRPO: an online reinforcement learning method that contrasts memory-on and memory-off rollouts to encourage selective use of ATMem, with memory-cost-aware reward.
  • A challenging mobile benchmark for evaluating whether agents complete all in-scope work while avoiding out-of-scope actions, with near-identical entries.
  • New metrics: App-Level Progress and Scope-Aware F1 to separately measure task progress and scope adherence.

Methodology

The paper proposes ATMem, a memory module that maintains a continually updated execution state of task-relevant information. Agents are first trained via supervised fine-tuning to construct ATMem, then fine-tuned with STR-GRPO, an online RL method that compares rollouts with and without memory to learn selective memory usage, using a reward that penalizes costly memory operations that do not improve execution. Evaluation is conducted on a newly built mobile benchmark requiring agents to act on entries matching instructions and reject others, with metrics App-Level Progress and Scope-Aware F1.

No specific experimental results are reported in the abstract; the paper introduces the benchmark and metrics for evaluation.
