Deep Analysis

Innovations

  • Native on-device agent framework for mobile phones that runs the full agent loop locally
  • Exposes device capabilities as tools with explicit arguments, structured results, and clearly defined execution boundaries, replacing long GUI action sequences
  • Open-source release of the framework

Methodology

PalmClaw is a mobile agent framework that runs natively on the device, managing sessions, memory, skills, tools, and the agent loop directly on the phone. It abstracts device capabilities as device tools with explicit arguments and structured results, enabling direct and controlled use of mobile functions instead of relying on GUI actions like tapping and swiping. Experiments compare PalmClaw against a strongest baseline (GUI-based approach) using metrics of task success and completion time.

Key Results

PalmClaw achieves an 11.5% relative improvement in task success and a 94.9% reduction in completion time over the strongest baseline, with lower setup burden.
