OpenWebRL: Demystifying Online Multi-turn Reinforcement Learning for Visual Web Agents

arXiv 2026 67.1 method, system

TLDR

OpenWebRL is an open framework for training visual web agents using online multi-turn reinforcement learning on live websites, achieving state-of-the-art results.

Reasoning

The paper addresses a key scalability bottleneck in supervised post-training by leveraging online RL on real websites, demonstrating strong empirical results on live-web benchmarks. Its strengths include a full training pipeline and competitive performance with proprietary systems, though it focuses solely on visual web agents and does not discuss limitations or generalization to other domains.

Deep Analysis

Innovations

  • OpenWebRL: an open framework for training visual web agents with online multi-turn RL on real websites, including scalable live-browser infrastructure, supervised initialization, multimodal context management, trajectory-level success judging, and efficient multi-turn policy optimization.
  • OpenWebRL-4B model that establishes new open-source state-of-the-art on live-web benchmarks, using only 0.4K initialization trajectories and 2.2K RL tasks.
  • Systematic study of design choices that make online RL effective for visual web agents and analysis of how RL improves agentic reasoning.

Methodology

The framework uses online multi-turn reinforcement learning on live websites, starting with supervised fine-tuning on 0.4K trajectories, then training a 4B-parameter model via multi-turn policy optimization with trajectory-level success judging. Evaluation is on Online-Mind2Web and DeepShop benchmarks.

Key Results

OpenWebRL-4B achieves 67.0% success on Online-Mind2Web and 64.0% on DeepShop, outperforming prior open agents of similar or larger scale and competitive with proprietary systems like OpenAI CUA and Gemini CUA.
