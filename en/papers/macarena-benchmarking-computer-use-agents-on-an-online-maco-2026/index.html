MacArena: Benchmarking Computer Use Agents on an Online macOS Environment

arXiv 2026 56.6 method

TLDR

MacArena is a benchmark of 421 macOS tasks across 50 apps, revealing that current GUI agents struggle more on macOS than Linux benchmarks suggest.

Reasoning

The paper addresses a clear gap in macOS benchmarking for computer-use agents, with a well-constructed benchmark combining ported and native tasks. Its strength lies in demonstrating that model rankings invert between task types, highlighting the need for cross-platform evaluation. A limitation is the focus solely on macOS, leaving generalizability to other OSes unaddressed.

Deep Analysis

Innovations

  • Introduces MacArena, a benchmark of 421 manually verified tasks across 50 applications for computer-use agents on macOS, combining ported OSWorld tasks, macOSWorld content, and 49 new macOS-native tasks.
  • Runs on Apple's native Virtualization framework on Apple Silicon, addressing the incompatibility of prior macOS benchmarks with Apple Silicon.
  • Demonstrates that macOS poses distinct GUI challenges and that model rankings invert between ported and macOS-native tasks, revealing that strong performance on existing benchmarks may reflect task distribution familiarity rather than cross-platform competence.

Methodology

The benchmark comprises 421 tasks spanning 50 applications, combining a curated port of OSWorld tasks, content from macOSWorld, and 49 new macOS-native tasks, all executed on Apple's native Virtualization framework on Apple Silicon. Computer-use agents are evaluated on this benchmark, and performance is compared between ported and macOS-native task subsets.

Key Results

Model rankings invert between ported and macOS-native tasks, with a leading model trailing by over 26% on the MacArena subset, indicating that macOS is a harder environment for current GUI agents.
