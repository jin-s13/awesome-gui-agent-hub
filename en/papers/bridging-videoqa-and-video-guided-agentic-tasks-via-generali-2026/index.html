Deep Analysis

Innovations

  • Introduction of VG-GUIBench, a benchmark for evaluating MLLM-based GUI agents on video-guided interactive tasks
  • Proposal of TASKER, a task-driven and scene-aware keyframe extraction algorithm that jointly considers task relevance and scene dynamics

Methodology

We introduce VG-GUIBench, a benchmark for video-guided GUI tasks, and propose TASKER, a keyframe extraction algorithm that selects frames based on task relevance and scene dynamics. MLLM-based agents are evaluated on both VideoQA and GUI interactive tasks using TASKER-extracted keyframes.

Key Results

TASKER outperforms the best baseline by 2.0% on EgoSchema fullset and 1.8% on NExT-QA for VideoQA, showing significant improvements from generalized keyframe extraction.
