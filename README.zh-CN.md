## 每日更新

GitHub Actions 会在 **UTC 00:00 / 北京时间 08:00** 每日刷新。
流水线会搜索近期论文、更新数据集、抓取 teaser 图片、补充 OpenAlex 引用元数据、
回填 TLDR / 深度分析字段、重建 Astro 站点并发布到 GitHub Pages。

必需的仓库 secret：`ARK_API_KEY`。
可选 secrets：`ARK_API_BASE_URL`、`ARK_MODEL_NAME`、`SMART_MODEL_NAME`、
`MINERU_API_KEY`、`OPENALEX_API_KEY`、`OPENALEX_MAILTO`、
`SEMANTIC_SCHOLAR_API_KEY`、`GH_DISCOVERY_TOKEN`。
可选仓库 variables：`ARK_TIMEOUT_SECONDS`、`ARK_SURVEY_TIMEOUT_SECONDS`、
`ARK_SURVEY_MAX_TOKENS`、`SEMANTIC_SCHOLAR_REQUEST_INTERVAL_SECONDS`。
workflow 也会把 GitHub 内置 token 作为 `GH_TOKEN` 传入，避免上游 awesome-list
发现流程退回匿名 GitHub Search 限流。
